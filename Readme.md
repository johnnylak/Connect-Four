
# Connect Four - CLI 

A CLS Connect 4 game written in C for Linux and Windows.

## Installing

Extract the zip file to any desired folder and make sure the vm, stock.dat and coin.dat are in the same folder.

## Usage

>gcc *.c -o run

>./run
